# `draggable`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

it is a boolean value indicating whether the slider should be draggable or not

## default

```typescript
draggable: true
```

<br />

### Example `draggable: true` (default)

```javascript
new BlazeSlider(el, {
  all: {
    draggable: true, // <--
    slidesToShow: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      draggable: true,
      slidesToShow: 3,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
</ExampleSlider>

<br />

### Example `draggable: false`

Now Try to drag the slider - You won't be able to drag it. This can be useful to programmatically control the slider

````javascript

```javascript
new BlazeSlider(el, {
  all: {
    draggable: false, // <--
    slidesToShow: 4,
  },
})
````

<ExampleSlider
