# `transitionDuration`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

The amount of time in milliseconds the slide should take to transition from position A to position B during autoplay.

## default

```typescript
transitionDuration: 500
```

<br />

### Example `transitionDuration: 300` (default)

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true,
    transitionDuration: 300, // (default)
    slidesToShow: 3,
    autoplayInterval: 2000,
  },
})
```

<ExampleSlider
  config={{
    all: {
      enableAutoplay: true,
      transitionDuration: 300,
      slidesToShow: 3,
      autoplayInterval: 2000,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
