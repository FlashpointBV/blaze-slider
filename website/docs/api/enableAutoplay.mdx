# `enableAutoplay`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

it is a boolean value indicating whether autoplay should be enabled or not.

## default

```typescript
enableAutoplay: false
```

<br />

### Example `enableAutoplay: true`

In this example, pagination and navigation buttons are also added, so that you can see the impact of the autoplay on them.

`stopAutoplayOnInteraction` is also turned off in this example (which is enabled by default) to keep the autoplay enabled.

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true, // <--
    slidesToShow: 3,
    stopAutoplayOnInteraction: false,
    autoplayInterval: 1500,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      slidesToShow: 3,
      enableAutoplay: true,
      stopAutoplayOnInteraction: false,
      autoplayInterval: 1500,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>

  <div> 6 </div>
</ExampleSlider>
